# 🤖 Machine Learning Model Comparison Project

This project demonstrates the implementation and comparison of multiple Machine Learning models on a dataset to evaluate their performance and understand which model works best.

---

## 📌 Objective

The goal of this project is to:

- Apply different ML algorithms on the same dataset
- Evaluate their performance using appropriate metrics
- Understand the difference between regression and classification models
- Build strong fundamentals of model evaluation

---

## 🧠 Models Implemented

### 🔹 Regression Model
- Linear Regression

### 🔹 Classification Models
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Decision Tree

---

## ⚙️ Workflow

1. Data Loading
2. Data Preprocessing
   - Feature Selection
   - Train-Test Split
   - Feature Scaling (StandardScaler)
3. Model Training
4. Prediction
5. Model Evaluation

---

## 📊 Evaluation Metrics

### 📈 For Regression:
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score
- Adjusted R² Score

### 📉 For Classification:
- Accuracy Score
- F1 Score
- Confusion Matrix

---

## 🔍 Key Learnings

- Importance of scaling in ML models
- Difference between regression and classification metrics
- Why F1-score is better for imbalanced data
- Role of Adjusted R² in feature selection
- Avoiding data leakage during preprocessing

---

## 🚀 Results Summary

- Linear Regression achieved ~70% R² score
- Adjusted R² indicated presence of some irrelevant features
- Classification models can be compared further to find optimal performance
